# UniFit - Milestones del Proyecto

### [M0] Milestone 0: Modelo del problema
- Modelo que recoja los conceptos clave del dominio del problema tratado en la [HU1](/docs/HU.md). El producto entregado se ha obtenido a partir de un correcto uso de la metodología, siguiendo las pautas para trabajar con Issues y el desarrollo progresivo del mismo nazca de la información aportada en la [HU1](/docs/HU.md). Se entregará en forma de código fuente.

---

### [M1] Milestone 1: Implementación de la lógica de negocio
- Código fuente encargado de implementar la lógica de negocio necesaria para obtener y estructurar la información sobre ejercicios, conforme al modelo definido en el milestone 0. El producto entregado resuelve el problema que describe la [HU1](/docs/HU.md), siendo verificado mediante el uso de test y habiendo seguido la metodología de desarrollo.
