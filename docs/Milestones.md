# UniFit - Milestones del Proyecto

### [M0] Milestone 0: Modelo del problema
- Modelo con los conceptos clave del dominio del problema tratado en la [HU1](/docs/HU.md) sobre la extracción de información de una web de entrenamientos, así como las diferentes entidades que forman parte del mismo, sus relaciones y los agregados necesarios para desarrollar el PMV. Se entregará en forma de código fuente.   

- Se considerará validado cuando el producto entregado se haya obtenido a partir de un correcto uso de la metodología, siguiendo las pautas para trabajar con Issues y el desarrollo progresivo del mismo nazca de la información aportada en la [HU1](/docs/HU.md).

---

### [M1] Milestone 1: Implementación de la lógica de negocio
- Módulo de código encargado de implementar la lógica de negocio necesaria para obtener y estructurar la información sobre ejercicios desde la página muscleandstrength.com, conforme al modelo definido en el milestone 0.

- Se considerará viable cuando el producto entregado resuelva el problema que describe la [HU1](/docs/HU.md), siendo verificado mediante el uso de test y habiendo seguido la metodología de desarrollo.
